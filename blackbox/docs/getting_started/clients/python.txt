===============================
Connecting to Crate with Python
===============================

Install
-------

Use `pip <https://pypi.python.org/pypi/pip>`__ to install the Crate
Python package:

.. code-block:: bash

    pip install crate

Import the client module from the ``crate`` package:

.. code-block:: python

    from crate import client

Connect to Crate
----------------

Use the ``client`` module to create a connection:

.. code-block:: python

    connection = client.connect(SERVER_IP)

Open a cursor to the database to enable queries.

.. code-block:: python

    cursor = connection.cursor()
