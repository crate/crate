======================
Install from a Tarball
======================

Latest Stable Release
---------------------

.. Version {{ CONFIG.stable\_version }}
   Released on {{ CONFIG.stable\_version\_date }}

`Download latest stable release`_.

Start Crate by extracting the downloaded archive, move into the *bin*
folder and run:

.. code-block:: bash

    ./crate

To create a cluster of Crate instances, run the command multiple times.
