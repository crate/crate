=================
3rd Party Plugins
=================

In addition to the :doc:`crate-plugins` crate also bundles some 3rd party
plugins.

elasticsearch-head
------------------

A web front end for an Elasticsearch cluster

:Endpoint: http://localhost:9200/_plugin/head

:Website: http://mobz.github.io/elasticsearch-head/

bigdesk
-------

Live charts and statistics for Elasticsearch cluster.

:Endpoint: http://localhost:9200/_plugin/bigdesk

:Website: http://bigdesk.org/


segmentspy
----------

Elasticsearch plugin to watch segment dynamics (additions, merges, deletes)

:Endpoint: http://localhost:9200/_plugin/segmentspy

:Website: https://github.com/polyfractal/elasticsearch-segmentspy

.. _Elasticsearch: http://www.elasticsearch.org
