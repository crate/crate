=============
Release Notes
=============

Information about individual CrateDB releases, typically including upgrade
information and changelog.

.. NOTE::

   The latest stable, testing, and nightly version is always available from the
   CrateDB download page.

   Older version of CrateDB are available in `the release archives`_.
